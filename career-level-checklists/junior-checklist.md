# Junior Engineer Checklist (2025)

Tick these and you’re no longer “just a coder”.

| Category                     | Must-Know Frameworks / Skills                                      | ✓ |
|------------------------------|--------------------------------------------------------------------|---|
| Code Quality                 | SOLID principles (understand all 5)                                |   |
|                              | Clean Code basics (meaningful names, small functions)              |   |
| Version Control              | Git: branching, rebasing, resolving conflicts                      |   |
|                              | Trunk-based or Git Flow (know both)                                |   |
| Architecture Basics          | Can read and explain a C4 Context + Containers diagram            |   |
|                              | Knows what MVC, Layered, and Client-Server architectures are      |   |
| Testing                      | Writes unit tests (JUnit, PyTest, Go testing, etc.)                |   |
|                              | Understands TDD flow (Red → Green → Refactor)                      |   |
| Process                      | Comfortable with Agile + daily standups                            |   |
|                              | Can pick and complete tickets from Jira/Linear                     |   |
| CI/CD                        | Has triggered a pipeline (GitHub Actions, GitLab CI, Jenkins)     |   |
| APIs                         | Can design and document a simple REST API (OpenAPI basics)        |   |
| Observability                | Adds meaningful logs and knows where to find them                  |   |
| Communication                | Writes clear PR descriptions and comments                          |   |

~ 0–2 years experience target.  
When you have 90 %+ checked → you’re ready for mid-level roles.
